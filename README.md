# Hello Word 跟世界說你好
## 遊戲設計風格
卡牌內容為9國語言之hello的說法，即"Hello" in nine different languages，故將遊戲名稱命名為「Hello Word」，。
使用國際主義設計風格(瑞士風格)搭配遊戲主題「跟世界說你好」，讓整體遊戲設計概念更有一致性。
## 遊戲方式
將該國家牌與該國語言之「你好」牌配對，計時翻完牌需花多久時間，時間越短者為最高分。
卡牌背面花色可由右上角設定按鈕選擇，共有六種設計。
